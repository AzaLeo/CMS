Администраторский раздел
- админраздел  доступен  только  для  пользователей с соответствующей ролью;
- на странице управления контентом выводится список существующих контент-блоков;
- реализована возможность создания, удаления и редактирования контент-блоков;
- контент-блоки состоят из следующих элементов: заголовок, отключение публикации, уровень доступа, отключение заголовка при выводе, определение позиции для показа, визуальный редактор для создания текста;
- при удалении контент-блоков необходимо подтверждение.

Меню
- для гостей показаны пункты: "Главная", "Регистрация", "Вход";
- для пользователей показаны пункты: "Главная", "Профиль", "Выход";
- для администраторов показаны пункты: "Главная", "Профиль", "Администраторский раздел", "Выход".

Лицевая часть
- определены две позиции (в зависимости от дизайна можно создавать еще);
- контент-блоки выводятся в свои позиции в виде текста и заголовка (если показ заголовка включен);
- в тексте применяется форматирование html-тегами (средствами редактора TinyMCE).
